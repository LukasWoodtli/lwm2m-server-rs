# lwm2m-server-rs

A lightweight LwM2M (Lightweight Machine-to-Machine) server and bootstrap-server implementation in Rust.

## Features

- **LwM2M Server** – Handles device registration and management
- **Bootstrap Server** – Device bootstrapping support (planned)
- **CBOR** – Compact Binary Object Representation for efficient data encoding (planned)
- **UDP Transport** – CoAP over UDP communication
- **Async** – Built on Tokio for async I/O


## Building

```sh
cargo build
```

## Running Tests

```sh
cargo test
```

## Documentation

```sh
cargo doc --open
```
